# Практическая работа №2: Основы PHP

Данная практическая работа включает в себя реализацию трех независимых веб-сервисов на PHP, использующих Docker для контейнеризации. Все сервисы разработаны с учетом разделения функциональности и повторного использования файлов.

## 1. Описание реализованных сервисов

### 1.1. Веб-сервис Drawer (Рисование SVG объектов)

Сервис для рисования SVG объектов, реализованный в директории `T1`.

*   **URL:** `http://localhost:8080/T1/index.php?id=<id_number>`
*   **Примеры использования:**
    *   Красный круг (значение по умолчанию): `http://localhost:8080/T1/index.php`
    *   Красный круг: `http://localhost:8080/T1/index.php?id=1`
    *   Синий прямоугольник: `http://localhost:8080/T1/index.php?id=2`
    *   Зеленый круг: `http://localhost:8080/T1/index.php?id=3`
    *   Желтый прямоугольник: `http://localhost:8080/T1/index.php?id=4`
    *   Фиолетовый круг (больший размер): `http://localhost:8080/T1/index.php?id=5`

*   **Файлы в `T1/`:**
    *   `index.php`: Основной скрипт, обрабатывающий запрос `id` и вызывающий генератор SVG.
    *   `svg_generator.php`: Содержит логику для декодирования `id` в параметры фигуры (форма, цвет, размер) и генерации соответствующего SVG-кода.

### 1.2. Веб-сервис Сортировка (Quick Sort)

Сервис для сортировки чисел с использованием алгоритма быстрой сортировки (Quick Sort), реализованный в директории `T2`.

*   **URL:** `http://localhost:8080/T2/index.php?arr=<comma_separated_numbers>`
*   **Пример использования:**
    *   `http://localhost:8080/T2/index.php?arr=5,2,9,1,5,6,3`
    *   `http://localhost:8080/T2/index.php?arr=10,3,8,1,7,4,2,9,6,5`

*   **Файлы в `T2/`:**
    *   `index.php`: Основной скрипт, принимающий входные данные, вызывает функцию сортировки и отображает результаты.
    *   `quick_sort.php`: Реализация алгоритма быстрой сортировки.

### 1.3. Веб-сервис Информация о сервере

Сервис для отображения информации о сервере, реализованный в директории `T3`.

*   **URL:** `http://localhost:8080/T3/index.php`
*   **Выполняемые команды:**
    *   `ls -la /var/www/html`
    *   `ps aux | head -n 5`
    *   `whoami`
    *   `id`
    *   `uname -a`
    *   `df -h /`
    *   `free -h`

*   **Файлы в `T3/`:**
    *   `index.php`: Основной скрипт, который вызывает функции выполнения команд и отображает их вывод.
    *   `system_command_executor.php`: Содержит безопасную функцию для выполнения произвольных Unix-команд.
